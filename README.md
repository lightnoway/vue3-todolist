交互-添加

交互-编辑
  - 双击item ，进入编辑状态
    - 切换显示：显示输入框，值为原文本值；隐藏原 check，label
    - 编辑时按Esc ，cancel: 直接回到（丢弃编辑的输入框中文本）非编辑态
    - 编辑时按回车，confirm: 
        - 输入框文本为空，删除当前item
        - 输入框文本不为空
            - 修改当前item.text
            - 回到非编辑态
